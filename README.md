# Pitch Detector
<!-- ![Build and Test](https://github.com/swe-students-fall2025/4-containers-national-team/actions/workflows/build.yaml/badge.svg) -->

**Pitch Detector** is a website to help you recognize the pitch of your voice using AI to help you analyze your vocal performances!

---

## How to install and use this package
1. **Run Docker Desktop**

2. **Build Docker**
```bash
docker compose build
```
3. **Run Docker Container**
```bash
docker compose up
```
4. **Access website!**
http://127.0.0.1:5000
