# Pitch Detector
<!-- ![Build and Test](https://github.com/swe-students-fall2025/4-containers-national-team/actions/workflows/build.yaml/badge.svg) -->

**Pitch Detector** is an application designed to recognize the pitch of your voice or instruments using machine learning to help you find the right tunes!

---

## Installation and Usage

If you'd like to install and use this app, first clone the repository:

    git clone https://github.com/swe-students-fall2025/4-containers-national-team.git

Once in the directory, use Docker Desktop to build and run. If you don't have Docker Desktop installed, install it first.

    docker compose build
    docker compose up

Finally, you can access the site at http://127.0.0.1:5000. To stop running, simply do:
    
    docker compose down
